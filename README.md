# Hoyo
Hoyo Packages
- 简化MongoDB驱动的使用
- 一些常用的工具扩展
- 类似ABP框架的自动模块注入
- 对IdentityServer 6 的Mongo存储支持
- MongoDB GridFS使用简化支持.
- 一些WebAPI中常用的过滤器,中间件的集成
